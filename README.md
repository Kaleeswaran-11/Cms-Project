# Django CMS Project

A full stack Content Management System (CMS) built using Django. This project includes user authentication, CRUD operations, dashboard functionality, and a responsive UI using Bootstrap.

---

## Features

- User Authentication (Login / Logout / Registration)
- Create, Read, Update, Delete (CRUD) functionality
- Admin Dashboard
- Role-based access control (basic)
- Responsive UI using Bootstrap
- Form validation and secure data handling

---

## Tech Stack

- Backend: Python, Django
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Database: SQLite / MySQL
- Tools: Git, GitHub

## How to Run

```bash
git clone https://github.com/YOUR_USERNAME/Cms-Project.git
cd Cms-Project

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
Open in browser:
http://127.0.0.1:8000/
